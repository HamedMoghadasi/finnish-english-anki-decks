# Finnish to English Words 
A curated collection of Finnish to English vocabulary files for language learners, organized by CEFR level. 

## Generated With
These word lists were generated using this tool:

[HamedMoghadasi/anthropic-anki-deck-generator](https://github.com/HamedMoghadasi/anthropic-anki-deck-generator)

## Overview
This repository contains topic-based Finnish vocabulary sets with English meanings and example sentences.
The files are suitable for study workflows and Anki import.

## Example file names:

Finnish Words - A1 - 016 - family relatives relationships.txt
Finnish Words - B2 - 008 - artificial intelligence ethics.txt
File Content Format
Most files are tab-separated text files.

## Typical columns:

word (Finnish)
meaning (English)
example (Finnish sentence)
example-meaning (English translation)
tags (optional, present in some generated files)
Some files may include Anki-compatible header lines at the top.

## Usage
Choose a level folder (A1 to B2).
Pick a topic file.
Import into your flashcard app (for example, Anki).
Map fields to your note type:
word
meaning
example
example-meaning
tags (if present)
Naming Convention
File names follow this pattern:

Finnish Words - LEVEL - NUMBER - TOPIC.txt

This keeps the dataset easy to search, sort, and expand.

## Who This Is For
Finnish learners who want organized vocabulary by level and topic
Teachers building themed lesson material
Self-learners creating spaced-repetition decks
Notes
Content focuses on practical, high-utility vocabulary.
Topics become more abstract and advanced at higher levels.
Tags can be used for filtered study sessions where available.
